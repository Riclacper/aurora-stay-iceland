# Aurora Stay Iceland

Plataforma web moderna para descoberta e reservas de hospedagens premium na Islândia, com foco em experiências exclusivas para observação da aurora boreal.

---

## Sobre o projeto

O Aurora Stay Iceland é um protótipo acadêmico desenvolvido com foco em experiência do usuário, interface moderna e arquitetura SPA (Single Page Application).

A aplicação simula uma plataforma profissional de reservas de hospedagens de luxo, incluindo:

- Página inicial interativa
- Sistema de busca inteligente
- Catálogo de hospedagens
- Página de detalhes
- Dashboard administrativo
- Navegação SPA
- Layout responsivo
- Design premium inspirado em plataformas reais

## Instalação

Clone o projeto:

```bash
git clone https://github.com/seuusuario/aurora-stay-iceland.git
```

Acesse a pasta:

```bash
cd aurora-stay-iceland
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

---

## Build de produção

```bash
npm run build
```

Preview local:

```bash
npm run preview
```
